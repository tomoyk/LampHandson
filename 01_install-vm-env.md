# 01.仮想環境と仮想マシンの構築

## 概要

Linuxの環境をWindows上にインストールした仮想化ソフトウェア(VMWare Workstation Player)によって構築します。

実機へのインストール（デュアルブート）は操作ミスによりWindows環境を破壊する恐れがあり、授業への支障を考え断念しました。
仮想環境であれば仮に操作ミスをしても仮想マシンの再作成で済むため影響範囲を最小限に抑えることができるため採用しています。
